このサイト「ファミマ行脚」は、私(@PEltptlt)が これまでに訪れ、買い物をしたファミリーマート各店舗の記録をまとめ、地図上で可視化した個人アーカイブです。

本サイトおよび私の活動は、ファミリーマート公式とは無関係の個人的なものです。

店舗記録加算条件

必須項目

・自身がレジに立つこと。(有人・セルフ不問)
・自身もしくは生計を同一とする家族が決済すること。
・レシートを受け取ること、家まで持ち帰ること。

努力項目

・自身のTカードおよびファミペイアプリをかざすこと。

店舗動態

記録をつける店舗名はレシートに書かれている店舗名とする。

・レシート紛失
記録しない、再訪問すれば通常通り記録する。
例：ファミリーマート 阿倍野二丁目店

・閉店
訪問済みの記録は減算しない。
例：ファミリーマート 二条聚楽町店

・店舗名変更
変更後の再訪問は数に加算しないが、変更前店舗の記録に補足を記す。
例：ファミリーマート 新幹線博多駅ホーム店 → 銘品蔵博多駅新幹線ホーム店
・施設名変更による店舗名変更
例：ファミリーマート 梅田ＯＰＡ店 → ＡＢＣ－ＭＡＲＴ梅田ビル店
・駅名変更による店舗名変更
例：ファミリーマート 浜大津駅前店 → びわこ浜大津駅前店

・改装もしくは、閉店した同じ場所に再開店
店舗名が同じであれば再訪問しない。
店舗名が異なる場合は、店舗名変更と同じ扱いとする。
例：ファミリーマート 河原町御池店 → 本能寺店

・かつて閉店した店舗と新しく開店した店舗で店舗名が一緒だが場所が違う
別店舗扱いで記録する。
例：ファミリーマート 河原町三条店
( 2015/08/31閉店 京都朝日会館1F入居店舗
→2017/11/16開店 河原町三条ビル1F入居店舗)

・サテライト店舗のためレシート表記の店舗名が母店
母店訪問済みの場合は母店訪問記録の補足にサテライト店名を記す。
例：ファミリーマート 近鉄鵜方駅改札外橋上店 → 志摩グリーンアドベンチャー／Ｎ店

母店未訪問の場合は母店訪問記録として新しく記録し、サテライト店名は補足に記す。
例：ファミリーマート 丸の内オアゾ店 → ＦＡＭＩＭＡ　ｆａｃｔｏｒｙ店

・サテライト店舗だがレシート表記の店舗名がサテライト店表記、後に母店が閉店し店舗名変更
サテライト店表記で登録(レシート表記通り)、補足に変更後店名を記す。 例：ファミリーマート ＴＳＵＴＡＹＡ長尾（サ）店 → ロームシアター京都店
