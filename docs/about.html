このサイト「ファミマ行脚」は、私(@PEltptlt)が これまでに訪れ、買い物をしたファミリーマート各店舗の記録をまとめ、地図上で可視化した個人アーカイブです。

本サイトおよび私の活動は、ファミリーマート公式とは無関係の個人的なものです。

店舗記録加算条件

必須項目

  • 自身がレジに立つこと。(有人・セルフ不問)
  • 自身もしくは生計を同一とする家族が決済すること。
  • レシートを受け取り、家まで持ち帰ること。

努力項目

  • 自身のTカードおよびファミペイアプリをかざすこと。

店舗動態

記録をつける店舗名はレシートに書かれている店舗名とする。
住所は現在、位置情報よりgoogle map APIを用いたリバースジオコーディングで取得したものを表記しており、
建物名等不正確なところがあります。後々なんとかしたい。

レシート紛失
記録しない。再訪問すれば通常通り記録する。
例：ファミリーマート 阿倍野筋二丁目店
閉店
訪問済みの記録は減算しない。
例：ファミリーマート 二条聚楽町店
店舗名変更
変更後の再訪問は数に加算しないが、変更前店舗の記録に補足を記す。
例：ファミリーマート 新幹線博多駅ホーム店 → 銘品蔵博多駅新幹線ホーム店
・施設名変更による店舗名変更
　例：ファミリーマート 梅田OPA店 → ABC-MART梅田ビル店
・駅名変更による店舗名変更
　例：ファミリーマート 浜大津駅前店 → びわこ浜大津駅前店
改装または閉店後の同一場所での再開店
・店舗名が同じであれば再訪問しない。
・店舗名が異なる場合は、店舗名変更と同じ扱いとする。
例：ファミリーマート 河原町御池店 → 本能寺店
閉店店舗と新規店舗で店舗名が同じだが住所が異なる場合
別店舗扱いで記録する。
例：ファミリーマート 河原町三条店
(2015/08/31閉店：京都朝日会館1F → 2017/11/16開店：河原町三条ビル1F)
サテライト店舗でレシート表記が母店の場合
・母店訪問済みの場合：母店訪問記録の補足にサテライト店名を記す。
・母店未訪問の場合：母店訪問記録として新規記録し、補足にサテライト店名を記す。
例：ファミリーマート 近鉄鵜方駅改札外橋上店 → 志摩グリーンアドベンチャー／N店
例：ファミリーマート 丸の内オアゾ店 → FAMIMA factory店
サテライト店舗だがレシート表記がサテライト店名で、後に母店が閉店し名称変更
レシート表記通りサテライト店として登録し、補足に変更後店名を記す。
例：ファミリーマート TSUTAYA長尾（サ）店 → ロームシアター京都店
サテライト店舗だがレシート表記の住所が母店と異なる場合
別店舗扱いで記録する。
例：ファミリーマート 新潟八千代二丁目店
（後にサテライト店は新潟バスセンター／S店に名称変更、その後移転）
